# GitCommand

## git 常用命令随记

* 常见分支`git branch 分支名`

* 切换分支`git checkout 分支名`

* 创建并切换`git checkout -b 分支名`

* 将内容上传暂存区`git add .`

* 撤销暂存区的内容`git rest HEAD`

* 暂存工作现场`git stash`

* 将暂存区的内容提交到本地仓库`git commit -m '注释'`

* 将本地仓库提交到远程仓库`git push origin 分支名`

* 拉取远程分支`git fetch origin 分支名`

* 拉取全部分支`git fetch`
